# 발급받은 키로 웹소켓으로 데이터 수집을 하면서 주문을 넣을수 있나?

## 웹소켓 데이터 수집과 주문 동시 가능 여부
키움증권 RESTAPI에서는 발급받은 접근토큰으로 웹소켓을 통해 실시간 시세 데이터를 수집하면서, 동일 토큰으로 주문(매수, 매도, 정정, 취소)도 가능합니다. 즉, 하나의 접근토큰으로 실시간 데이터 수신과 주문 기능을 동시에 사용할 수 있습니다.

### ✅ 요약
> 키움증권 RESTAPI는 하나의 접근토큰으로 웹소켓 실시간 시세 수신과 REST API 주문 요청을 동시에 지원합니다.  
> 실전투자의 경우 KRX, NXT, SOR 거래소 모두 지원하며, 웹소켓과 주문 API를 병행하여 사용할 수 있습니다.  
> Python 환경에서 웹소켓 연결 후 실시간 데이터 수신과 별도의 HTTP 요청으로 주문을 처리하는 방식입니다.

### 📘 설명
1 웹소켓 연결 시 접근토큰으로 로그인하여 실시간 시세 데이터를 수신합니다.
2 동일한 접근토큰을 HTTP 헤더에 포함하여 주문 관련 REST API를 호출할 수 있습니다.
3 실시간 데이터 수신과 주문 요청은 별도의 통신 채널(웹소켓과 HTTP)로 처리되므로 충돌 없이 병행 가능합니다.
4 실전투자에서는 KRX, NXT, SOR 거래소 모두 주문 가능하며, 모의투자는 KRX만 지원합니다.
5 주문 API 예시: 주식 매수(fn_kt10000), 매도(fn_kt10001), 정정(fn_kt10002), 취소(fn_kt10009) 등이 있습니다.
6 웹소켓과 주문 API 모두 동일한 접근토큰을 사용하므로, 토큰 관리에 유의해야 합니다.

### Input / Output 규격 관련
- Input(WebSocket): 접근토큰 (로그인용)
- Input(주문 API): 접근토큰 (authorization 헤더), 주문 관련 파라미터(종목코드, 수량, 가격 등)
- Output(WebSocket): 실시간 시세 데이터 JSON
- Output(주문 API): 주문 결과 JSON (주문번호, 상태 등)

import asyncio
import websockets
import json
import requests

SOCKET_URL = 'wss://api.kiwoom.com:10000/api/dostk/websocket'
ACCESS_TOKEN = '사용자 AccessToken'  # 변수화 필수

class WebSocketClient:
    def __init__(self, uri):
        self.uri = uri
        self.websocket = None
        self.connected = False
        self.keep_running = True

    async def connect(self):
        try:
            self.websocket = await websockets.connect(self.uri)
            self.connected = True
            login_packet = {'trnm': 'LOGIN', 'token': ACCESS_TOKEN}
            await self.send_message(login_packet)
        except Exception as e:
            print(f'Connection error: {e}')
            self.connected = False

    async def send_message(self, message):
        if not self.connected:
            await self.connect()
        if self.connected:
            if not isinstance(message, str):
                message = json.dumps(message)
            await self.websocket.send(message)

    async def receive_messages(self):
        while self.keep_running:
            try:
                response = json.loads(await self.websocket.recv())
                if response.get('trnm') == 'LOGIN':
                    if response.get('return_code') != 0:
                        print('로그인 실패:', response.get('return_msg'))
                        await self.disconnect()
                    else:
                        print('로그인 성공')
                elif response.get('trnm') == 'PING':
                    await self.send_message(response)
                else:
                    print('실시간 데이터 수신:', response)
            except websockets.ConnectionClosed:
                print('서버 연결 종료')
                self.connected = False
                await self.websocket.close()

    async def run(self):
        await self.connect()
        await self.receive_messages()

    async def disconnect(self):
        self.keep_running = False
        if self.connected and self.websocket:
            await self.websocket.close()
            self.connected = False

async def main():
    ws_client = WebSocketClient(SOCKET_URL)
    receive_task = asyncio.create_task(ws_client.run())

    await asyncio.sleep(1)
